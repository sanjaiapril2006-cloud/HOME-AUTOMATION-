# HOME-AUTOMATION-
Title: Smart Home Automation System  Objective: Automate home monitoring and control.  Components: Arduino Uno, DHT22, LDR, PIR Sensor, LCD, LEDs.  Working: Controls lighting based on ambient light, detects motion for security, and displays temperature and sensor data on an LCD.  for reference visit : https://wokwi.com/projects/467357567721788417
